Funcionalidades desenvolvidas: Foram adicionadas novas ferramentas do codigo java (List, exceções,
atributos private, uso de get's e set's para acessalos, novas interfaces e utilização de herança)

Orientações para compilação e execução : Executar javac Main.java e logo em seguida java Main

Descrição como realizar as operações presentes do sistema: Para bom uso do sistema, deve-se ter
pelo menos 3 Profissionais de 3 areas diferentes, com todas as informações solicitadas. 
Após isso podemos casdastra os pacientes e realizar consultas para os profissionais cadastrados 
anteriomente. Caso algum erro tenha sido cometido ao cadastrar paciente ou profissional, é 
possivel atualizar esses dados, como também listar todos eles em uma lista completa e 
fazer uma pesquisa de acordo com um criterio para profissional (Tipo de profissional  )
